Integritetspolicy

Här förklaras hur Nordlöf Nordic behandlar personuppgifter via webbutvecklare.se. Kort sagt: jag samlar bara in sådant som behövs för att kunna svara, hjälpa dig, ta fram demo, lämna offert eller genomföra uppdrag inom hemsidor, webbfix och webbutveckling.

Läs policyn Kontakta mig

Kort version

När du kontaktar mig via formulär, e-post eller telefon behandlas dina uppgifter för att jag ska kunna svara, hantera din förfrågan och vid behov genomföra ett uppdrag. Uppgifter säljs inte vidare och sparas inte längre än det finns ett rimligt skäl. Vissa tekniska tjänster kan användas för formulär, e-post, hosting, indexering och företagsadministration. Inget mystiskt datamonster i garderoben.

1. Personuppgiftsansvarig

Personuppgiftsansvarig för behandling av personuppgifter via webbutvecklare.se är Nordlöf Nordic, med Erica Nordlöf som kontaktperson.

Kontakt vid frågor om personuppgifter: erica@webutvecklare.se.

2. Vilka uppgifter som kan behandlas

När du kontaktar Nordlöf Nordic via formulär, e-post eller telefon kan följande uppgifter behandlas, beroende på vad du själv lämnar:

  • ✓Namn
  • ✓E-postadress
  • ✓Telefonnummer
  • ✓Företagsnamn eller verksamhetsnamn
  • ✓Webbadress till nuvarande hemsida
  • ✓Information om din förfrågan, brief, budget, önskemål eller uppdrag
  • ✓Material du själv skickar, till exempel text, bilder, länkar, dokument eller inspiration
  • ✓Teknisk information som behövs för att bygga, uppdatera eller lämna över en hemsida eller webblösning

Skicka helst inte känsliga personuppgifter via formulär eller vanlig e-post om det inte verkligen behövs. En bra brief behöver inte bli en självbiografi med känsliga bilagor.

3. Varför uppgifterna används

Personuppgifter används för att kunna:

  • ✓Svara på frågor och förfrågningar
  • ✓Ta fram demo, offert, prisförslag eller upplägg
  • ✓Genomföra beställda uppdrag
  • ✓Kommunicera om pågående arbete, ändringar, leverans och betalning
  • ✓Hantera administration, bokföring och affärsdokumentation
  • ✓Lämna över kod, filer, material och åtkomstuppgifter efter köp när det behövs

Uppgifterna används alltså för praktiskt kundarbete och normal företagsadministration. Inte för annonseringscirkus eller onödig profilering.

4. Rättslig grund

Behandlingen sker huvudsakligen med stöd av:

  • ✓Berättigat intresse, till exempel för att kunna svara på en förfrågan och hantera kontakt
  • ✓Avtal eller åtgärder inför avtal, till exempel när du ber om offert, demo eller beställer en tjänst
  • ✓Rättslig förpliktelse, till exempel när uppgifter behöver sparas för bokföring

Om någon framtida behandling kräver samtycke används samtycke för just den delen.

5. Formulär, e-post och telefon

När du skickar ett formulär via webbutvecklare.se skickas informationen vidare till Nordlöf Nordic via den formulärtjänst som används på sidan. Formuläret används för att ta emot förfrågningar om till exempel gratis demo, ny hemsida, webbfix, landningssida, formulärflöde, API eller webbutveckling.

När du mejlar eller ringer behandlas uppgifter som behövs för att kunna svara och hantera kontakten. Om du blir kund kan kommunikationen även sparas som del av uppdragets underlag.

Vill du minimera mängden information i första kontakten går det bra att skriva kort först. Resten kan tas i nästa steg när uppdraget är tydligare.

6. Cookies, lokal lagring och analys

Webbutvecklare.se använder inte annonseringscookies och har inte Google Analytics inlagt i denna version. Google Analytics är därför inte en del av den nuvarande standardmätningen.

Google Search Console och Bing Webmaster Tools kan användas för teknisk synlighet, indexering och felsökning i sökresultat. Dessa verktyg används för att förstå hur webbplatsen syns i sök, inte för att följa enskilda besökare i ett reklamflöde.

Sidan kan använda externa resurser, till exempel Google Fonts, för typsnitt. Det kan innebära att teknisk information, som IP-adress och webbläsarinformation, behandlas av extern leverantör när sidan laddas.

Om externa länkar används, till exempel till Google Maps, LinkedIn eller GitHub, gäller respektive extern tjänsts egna villkor och integritetspolicy när du klickar vidare.

7. Delning av uppgifter och använda verktyg

Personuppgifter säljs inte vidare. Uppgifter delas inte heller i onödan. Däremot kan vissa tjänster behövas för att webbplatsen, kommunikationen och företagets administration ska fungera.

  • ✓Formulärtjänst för att ta emot formulärmeddelanden
  • ✓E-posttjänst för att läsa och besvara förfrågningar
  • ✓Webbhotell, domänleverantör eller publiceringsmiljö
  • ✓Faktura-, betal- eller bokföringstjänster när uppdrag genomförs
  • ✓Google Search Console och Bing Webmaster Tools för indexering och teknisk synlighet

Vid kunduppdrag kan teknisk information också behandlas om det krävs för att bygga, uppdatera, publicera eller lämna över hemsidan eller webblösningen.

8. Hur länge uppgifter sparas

Uppgifter sparas inte längre än nödvändigt för det ändamål de samlades in för.

  • ✓Förfrågningar som inte leder till uppdrag sparas normalt bara så länge det behövs för kontakt och uppföljning
  • ✓Uppgifter kopplade till genomförda uppdrag kan sparas längre för administration, support, leverans, överlämning eller bokföring
  • ✓Bokföringsunderlag sparas enligt de krav som gäller enligt lag

Om du vill fråga vad som finns sparat eller be om radering där det är möjligt, kontakta erica@webutvecklare.se.

9. Dina rättigheter

Du har enligt dataskyddsreglerna rätt att i vissa fall:

  • ✓Få information om vilka personuppgifter som behandlas
  • ✓Begära rättelse av felaktiga uppgifter
  • ✓Begära radering av uppgifter när det är möjligt
  • ✓Invända mot viss behandling
  • ✓Begära begränsning av behandling i vissa situationer

Du kan kontakta Nordlöf Nordic via e-post om du vill använda någon av dina rättigheter. Du har också rätt att lämna klagomål till Integritetsskyddsmyndigheten om du anser att personuppgifter behandlas fel.

10. Säkerhet

Rimliga tekniska och organisatoriska åtgärder används för att skydda personuppgifter mot obehörig åtkomst, förlust eller missbruk.

Samtidigt är ingen vanlig webbplats, e-post eller formulärtjänst en magisk bunker. Skicka därför inte mer information än vad som behövs för uppdraget.

Om inloggningsuppgifter, filer eller annat känsligare material behöver delas i ett uppdrag bör det göras på ett så säkert och avgränsat sätt som möjligt.

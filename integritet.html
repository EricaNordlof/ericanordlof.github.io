Integritetspolicy utan datadimma.

Kort version

Jag samlar in det som krävs för att kunna hjälpa dig.

När du skickar formulär, mejlar eller ringer behandlas de uppgifter som behövs för att svara, hantera din förfrågan och vid behov genomföra uppdraget.

  • Kontaktuppgifter används för att kunna återkomma
  • Projektinformation används för demo, offert och leverans
  • Fakturauppgifter sparas enligt bokföringskrav
Dataminimering

Skicka inte mer än uppdraget behöver.

En bra brief behöver inte vara en självbiografi med känsliga bilagor. Skicka helst bara den information som behövs för att jag ska förstå uppdraget och kunna svara.

  • Undvik känsliga uppgifter i formulär
  • Dela inloggningar på säkert sätt vid behov
  • Håll första kontakten enkel

2. Uppgifter

Vilka uppgifter som kan behandlas.

När du kontaktar Nordlöf Nordic via formulär, e-post eller telefon kan uppgifter behandlas beroende på vad du själv lämnar.

Exempel

Vanliga uppgifter.

  • Namn
  • E-postadress
  • Telefonnummer
  • Företagsnamn eller verksamhetsnamn
  • Webbadress till nuvarande hemsida
  • Brief, budget, material, länkar och uppdragsbeskrivning
  • Bilder, texter, dokument eller inspiration som du själv skickar
  • Teknisk information som krävs för att bygga, uppdatera eller lämna över en webbplats

Leverans

Uppgifter används för kunddialog, ändringar, publicering, överlämning, support, betalning och bokföring.

Berättigat intresse

5. Formulär

Formulär, e-post och telefon.

När du skickar ett formulär via Webutvecklare.se skickas informationen vidare till Nordlöf Nordic via den formulärtjänst som används på sidan.

Formulär används för att ta emot förfrågningar om till exempel gratis demo, ny hemsida, webbfix, landningssida, formulärflöde, API eller webbutveckling.

När du mejlar eller ringer behandlas uppgifter som behövs för att kunna svara och hantera kontakten. Om du blir kund kan kommunikationen även sparas som del av uppdragets underlag.

Praktiskt tips

Första kontakten kan vara kort.

Vill du minimera mängden information i första kontakten går det bra att skriva kort först. Resten kan tas i nästa steg när uppdraget är tydligare.

  • Beskriv problemet
  • Länka gärna nuvarande sida
  • Skicka känsligt material först när det verkligen behövs

7. Delning

Delning av uppgifter och använda verktyg.

Personuppgifter säljs inte vidare. Uppgifter delas inte heller i onödan. Däremot kan vissa tjänster behövas för att webbplatsen, kommunikationen och företagets administration ska fungera.

Verktyg

Tjänster som kan användas.

  • Formulärtjänst för att ta emot formulärmeddelanden
  • E-posttjänst för att läsa och besvara förfrågningar
  • Webbhotell, domänleverantör eller publiceringsmiljö
  • Faktura-, betal- eller bokföringstjänster när uppdrag genomförs
  • Google Search Console och Bing Webmaster Tools för indexering och teknisk synlighet
8. Lagring

Hur länge uppgifter sparas.

Uppgifter sparas inte längre än nödvändigt för det ändamål de samlades in för. Kort sagt: så länge de behövs för kontakt, uppdrag, administration eller lagkrav.

Exempel

Lagring i praktiken.

  • Förfrågningar som inte leder till uppdrag sparas normalt bara så länge det behövs för kontakt och uppföljning
  • Uppgifter kopplade till genomförda uppdrag kan sparas längre för administration, support, leverans och överlämning
  • Bokföringsunderlag sparas enligt de krav som gäller enligt lag

10. Säkerhet

Rimliga skyddsåtgärder.

Rimliga tekniska och organisatoriska åtgärder används för att skydda personuppgifter mot obehörig åtkomst, förlust eller missbruk.

Samtidigt är ingen vanlig webbplats, e-post eller formulärtjänst en magisk bunker. Skicka därför inte mer information än vad som behövs för uppdraget.

Inloggningar

Dela åtkomst smart.

Om inloggningsuppgifter, filer eller känsligare material behöver delas i ett uppdrag bör det göras så säkert och avgränsat som möjligt.

  • Dela bara det som behövs
  • Byt lösenord efter avslutat arbete vid behov
  • Undvik att skicka onödigt känsligt material via vanlig e-post

Vanliga frågor om integritet.

Snabba svar, utan personuppgifts-sudoku.

Säljs mina uppgifter vidare?

Nej. Personuppgifter säljs inte vidare. De används för kontakt, demo, offert, uppdrag, leverans och nödvändig administration.

Använder sidan Google Analytics?

Nej, Google Analytics är inte inlagt i den nuvarande versionen av Webutvecklare.se.

Vad händer när jag skickar formulär?

Uppgifterna skickas via formulärtjänsten till Nordlöf Nordic så att jag kan svara på din förfrågan.

Kan jag be om radering?

Ja, du kan kontakta mig och begära radering där det är möjligt. Vissa uppgifter kan behöva sparas enligt lag, till exempel bokföringsunderlag.

Ska jag skicka lösenord via formulär?

Nej, undvik det. Om åtkomst behövs i ett uppdrag är det bättre att dela den på ett mer avgränsat och säkert sätt.

Nätverk · Infrastruktur · Automation

ResilientCore Lab

Ett självständigt nätverkslabb där jag utforskar hur ett redundant operatörsliknande nät kan designas, valideras, automatiseras och återhämta sig när en del av nätet går ner.

6/6 Noder operativa
2/2 VPNv4-sessioner
2 Friska tjänster
P2 Alternativ väg vid fel

Redundant nätarkitektur

Labbet använder en PE/P/CE-modell med dubbla core-vägar. Syftet är att visa hur nätet kan ha alternativa vägar om en länk eller nod får problem.

Routing & VPN-koncept

Projektet dokumenterar och demonstrerar koncept kring IS-IS, MPLS och Segment Routing, MP-BGP, L3VPN, VRF samt tjänstemodeller som EVPN, VPWS och VPLS.

Automation & validering

Python och Ansible används som delar av ett strukturerat arbetssätt för pre-check, validering, förändringskontroll och tekniska kontroller.

Simulerad incident

Dashboarden visar ett scenario där länken mellan PE1 och P1 försvinner. Trafiken ska då kunna använda den alternativa vägen via P2 och nätstatus verifieras efter förändringen.

Säkerhet & dokumentation

Projektet innehåller även resonemang kring segmentering, åtkomst, nätsäkerhet, förändringsplaner, rollback och tredje linjens felsökning.

Varför jag byggde det

För att utveckla min förståelse för hur nätverk, redundans, säkerhet och automation fungerar under applikationslagret. Det är ett portfolio-labb och ska inte förväxlas med professionell operatörsnätserfarenhet.

Fullstack · Blockchain · Web3

RentChain

Ett självständigt portfolio-case som visar hur bokningar kan verifieras kryptografiskt utan att kunddata behöver lagras på blockchain.

Lösning

Bokningsdata omvandlas till ett SHA-256-hashvärde som kan registreras och verifieras mot ett smart contract.

Tekniskt fokus

Next.js och TypeScript kombineras med Solidity, ethers.js, smart contracts och wallet-integration.

Integritet

Endast kryptografiska bevis behöver lagras on-chain. Person- och bokningsdata kan hållas privat off-chain.

Resultat

En driftsatt demo där flödet create → hash → register → verify går att testa direkt.

## 1. Offroad Bumpis Booking

Ett eget boknings- och administrationssystem byggt för en verklig uthyrningsverksamhet.

### Problemet

Verksamheten behövde hantera bland annat:

- bokningar
- datum och tillgänglighet
- olika typer av utrustning
- lagerregler
- priser
- flerdagarsbokningar
- rabattkoder
- leverans
- betalningsstatus
- dokument
- administration

Manuell hantering blir snabbt svår att skala och ökar risken för fel.

### Lösningen

Jag utvecklade ett eget boknings- och administrationsflöde anpassat efter verksamhetens faktiska regler.

Systemet hanterar bland annat:

- dynamisk tillgänglighet
- inventarie- och lagerregler
- barn- och vuxenutrustning
- flerdagarsbokningar
- rabattkoder
- leveransalternativ
- tillval
- betalningsstatus
- administrativa flöden
- dokument- och fakturaflöden
- responsivt gränssnitt

### Mitt ansvar

Jag har arbetat med hela kedjan:

- produktidé
- kravställning
- verksamhetslogik
- UX
- frontend
- backend
- datamodellering
- testning
- deployment
- löpande vidareutveckling

### Teknik

Projektet och dess relaterade system använder bland annat:

## 3. Safe Standard App

En React- och TypeScript-applikation för strukturerad dokumentation vid direktuthyrning.

### Problemet

Vid direktuthyrning kan information snabbt bli utspridd mellan:

- avtal
- sms
- mejl
- bilder
- egna villkor
- överlämning
- skadeinformation
- dokument

Det gör det svårt att hålla ett komplett och strukturerat underlag kring samma uthyrning.

### Lösningen

Safe Standard organiserar dokumentationen kring ett sammanhängande rental case.

Appen innehåller bland annat:

- Agreement
- Custom Terms
- Handover Checklist
- Damage Documentation
- Invoice / Damage Invoice
- Booking Link
- Bookings
- Documents
- Public Verification
- License Gate
- Dashboard
